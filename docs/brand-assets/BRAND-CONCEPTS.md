## I. THE RAVIO BRAND

### Origin

**RAVIO** = **R**ent-**A**-**V**acation **I**ntelligent **O**perator

RAVIO is the named AI personality behind all AI-powered features on the platform. It's not a chatbot — it's a concierge. It doesn't search — it understands. It doesn't filter — it finds.

### RAVIO Character Brief

| Attribute | Value |
|-----------|-------|
| **Name** | RAVIO |
| **Role** | AI Vacation Concierge |
| **Personality** | Knowledgeable, warm, efficient, slightly enthusiastic |
| **Voice** | Conversational, clear, helpful — like a well-traveled friend |
| **Capabilities** | Voice search, text chat, property recommendations, resort knowledge |
| **Limitations** | Honest about what it doesn't know. Never fabricates. |
| **Tagline** | "Just Say Where." |

### RAVIO in Marketing

**"Ask RAVIO"** — The voice search experience
- Usage: "Ask RAVIO to find your next vacation"
- Context: The primary AI interaction — voice-first, natural language
- Demo: Say "Find me a beachfront 2-bedroom in Maui under $2,000"
- Status: [BUILT]

**"Chat with RAVIO"** — The text search experience
- Usage: "Chat with RAVIO — type your dream vacation"
- Context: For users who prefer text, are in noisy environments, or have accessibility needs
- Demo: Type "I'm looking for a Disney resort with 3 bedrooms for a family reunion"
- Status: [BUILT]

## VIII. NAMING CONVENTIONS & USAGE RULES

### Always Capitalize
- Rent-A-Vacation (always hyphenated, title case)
- RAVIO (all caps when standalone; "Ask RAVIO" / "Chat with RAVIO" in feature context)
- Ask RAVIO, Chat with RAVIO
- Vacation Wishes
- Name Your Price
- RAV SmartPrice
- Fee Freedom Calculator
- TrustShield
- PaySafe
- ResortIQ
- RAV Command
- Owner's Edge
- Liquidity Score, Bid Spread Index

### Never
- "Rent a Vacation" (missing hyphens)
- "Ravio" (lowercase — always RAVIO)
- "Smart Price" (two words — always SmartPrice)
- "Trust Shield" (two words — always TrustShield)
- "Pay Safe" (two words — always PaySafe)

### Abbreviation
- **RAV** — acceptable in informal contexts, internal docs, and UI where space is limited
- **R-A-V** — never use dashes in abbreviation
